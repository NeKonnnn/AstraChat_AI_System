## [0.1.1.1] - 2026-02-03

### Added

- Разработаны абстрактные классы для применения настроек из config.yml в директории backend/settings. Созданы следующие файлы:
  - backend/settings/config.py - основной класс Settings для загрузки и управления конфигурацией из YAML файла
  - backend/settings/connections.py - абстрактные классы конфигурации подключений: MongoDBConnectionConfig, PostgreSQLConnectionConfig, MinIOConnectionConfig, LLMServiceConnectionConfig
  - backend/settings/__init__.py - экспорт всех классов и функций для удобного импорта
  - backend/settings/README.md - документация по использованию модуля настроек

Классы обеспечивают централизованное управление настройками подключений к внешним сервисам (MongoDB, PostgreSQL, MinIO, LLM Service) с поддержкой загрузки из YAML файла (backend/config/config.yml) и переменных окружения. YAML файл конфигурации находится в backend/config/config.yml. Все значения конфигурации вынесены в config.yml для удобного управления настройками приложения.
